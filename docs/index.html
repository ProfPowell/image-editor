<image-editor>

A basic image editor web component for cropping, rotating, filtering, resizing, and exporting images. Built with vanilla JavaScript and no dependencies.

Features

  • Crop with aspect ratio presets and draggable handles
  • Rotate 90° left/right and flip horizontal/vertical
  • Resize with aspect ratio lock and common presets
  • Filters: brightness, contrast, saturation, blur, grayscale, sepia
  • Undo/redo with configurable history depth
  • Export as PNG, JPEG, or WebP with quality control
  • Zoom and pan for detailed editing
  • Drag & drop or file picker image input
  • Light/dark mode with auto-detection
  • Full keyboard accessibility
  • 35+ CSS custom properties for theming
  • Vanilla Breeze compatible

Quick Start

Installation

npm install @profpowell/image-editor

Usage

<script type="module">
  import '@profpowell/image-editor'
</script>

<image-editor></image-editor>

With a pre-loaded image

<image-editor src="photo.jpg"></image-editor>

Dark mode with JPEG export

<image-editor mode="dark" format="jpeg" quality="0.85"></image-editor>
